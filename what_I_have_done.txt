Try running:
    containsElem 1 [1,2,3]
    Response: True

    containsElem 'a' "cde"
    Response: False

Reasoning: ContainsElem checks whether the variable is in a specific object,
above is an array, check whether 1 is in that array. Same goes with 'a' in the string cde.

Other examples:
    containsElem 5 [1,2,3,4,5]       -- True
    containsElem 0 [1,2,3]           -- False
    containsElem 'x' "hello world"   -- False
    containsElem 'o' "hello world"   -- True
    containsElem "hi" ["hi","bye"]   -- True

Discuss what the type specification means, containsElem :: Eq a =>
a-> [a]-> Bool. What is a and Eq?
    a is type variable
    Eq a is the type class that supports equality, == and /=
    a -> [a] -> Bool
        means that we take an element of type a,
        take a list of elements of type a
        and returns a bool and because of the function
        containing "containsElem" means that we check whether or not
        the element is in that list.
